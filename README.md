# PocketURL

**Big links, small pockets**. PocketURL is a lightweight URL shortening service
built as a demonstration project. It takes long, unwieldy URLs and "stashes"
them into compact, easy-to-share links.

# Features

- **Instant Shrinking**: Convert any URL into a pocket.url/alias in
  milliseconds.

- **Analytics**: Tracks how many times your pocketed links have been "unfolded."

- **Minimalist UI**: A clean, focused interface designed for speed.

# Tech Stack

- Frontend: NuxtJS

- Backend: Laravel Octane with FrankenPHP

- Database: MySQL, Postgres, Redis
